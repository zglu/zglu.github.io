Selected tools for single-cell and spatial transcriptomics.

scRNA-seq

  • ALRA: Zero-preserving imputation
  • HotSpot: identifies informative gene modules (pip install hotspotsc==0.9.1)
  • SCENIC/pySCENIC: Single-cell regulatory network and clustering
  • PROGENy & decoupleR: inference of pathway activities (human & mouse)
  • MetaNeighbor: characterizing the replicability of cell types across datasets (R)
  • SAMap: mapping cell atlas manifolds across species (py)
  • MAGIC: imputing missing values and restoring the structure of single-cell data (R & py)

spatial transcriptomics

  • Giotto: toolbox for integrative analysis of spatial data
  • SSAM (lite): segmentation-free inference of cell types
  • SpatialDE: identification of spatially variable genes (py)
  • Spark: Statistical analysis of spatial expression patterns
  • BayesSpace: Clustering and Resolution Enhancement of Spatial Transcriptomes
  • SpaGCN: spatial domains and SVGs by graph convolutional network
  • GCNG: GCN for inferring cell-cell interactions
  • SpaOTsc: spatial optimal transport for SC data (py)
  • DSSC: Deep Spatially constrained Single-cell Clustering (using spatial information and marker genes)
  • RCTD: Robust Cell Type Decomposition
  • C-SIDE: Cell type-specific inference of differential expression (spacexr = RCTD + C-SIDE; Vignettes)

Integration of single cell data

Benchmarking analysis

  • Benchmarking spatial and single-cell transcriptomics integration methods for transcript distribution prediction and cell type deconvolution

Selected tools

  • PASTE: Alignment and integration of spatial transcriptomics data from multiple adjacent tissue slices (Python)
  • GLUE: Multi-omics single-cell data integration and regulatory inference with graph-linked embedding (Python/R)
  • CellTrek: Spatial charting of single-cell transcriptomes in tissues (R)
  • Tangram: aligns sc/snRNA-seq data to spatial data (Python)
  • SPOTlight: integration of Spatial Transcriptomics with scRNA-seq data to infer the location of cell types and states (R)
  • Harmony: Fast, sensitive and accurate integration of single-cell data (R)
  • cell2location: Cell2location maps fine-grained cell types in spatial transcriptomics (Python)

Automated cell type annotation Turorial

Reference-based

  • RCTD/spacexr: Robust decomposition of cell type mixtures in spatial transcriptomics (R)
  • scmap: unsupervised projection of scRNA-seq data (R)
  • SingelR: Reference-based cell type annotation
  • CelliD: clustering-free extraction of per-cell gene signatures (R)
  • ScType: fully-automated and ultra-fast cell-type identification (R)
  • SciBet: Single Cell Identificator Based on E-test (R)

Marker-based

  • AUCell: cells with active gene sets
  • GSEA/GSVA: gene set enrichment analysis (test) (for clusters)
  • SCINA: Semi-Supervised Cell Type Detection and Assignment (R) (for cells)
  • MACA: marker-based automatic cell-type annotation / ScTypeDB: cell-type marker database

Knowledge-based clustering and annotation

  • UNIFAN: using known gene sets + neural network
